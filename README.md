# Sistema de Gerenciamento de Medicamentos – TCC

Projeto desenvolvido como Trabalho de Conclusão de Curso (TCC) do curso
Tecnologia em Sistemas para Internet.

O sistema tem como objetivo auxiliar no gerenciamento de medicamentos,
promovendo o descarte correto, o reaproveitamento consciente e a
sustentabilidade, além de facilitar o controle e a organização.

## Tecnologias utilizadas
- PHP
- Laravel
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap

## Funcionalidades
- Cadastro e gerenciamento de medicamentos
- Alertas de vencimento
- Controle de estoque
- Cadastro e gerenciamento de usuários
- Registro de doações e reaproveitamento de medicamentos
- Relatórios básicos

## Arquitetura do sistema
- Controllers responsáveis pela lógica da aplicação
- Models para manipulação e acesso aos dados
- Migrations para criação e versionamento do banco de dados
- Views desenvolvidas com Blade para interface do usuário
