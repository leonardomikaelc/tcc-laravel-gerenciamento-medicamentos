# Sistema de Gerenciamento de Medicamentos – TCC

Projeto desenvolvido como Trabalho de Conclusão de Curso (TCC) do curso
Tecnologia em Sistemas para Internet.

## Tecnologias utilizadas
- PHP
- Laravel
- MySQL
- HTML, CSS, JavaScript
- Bootstrap

## Funcionalidades
- Cadastro e gerenciamento de medicamentos
- Alertas de vencimento
- Controle de estoque
- Cadastro de usuários
- Relatórios básicos

## Objetivo
Promover o descarte correto e o reaproveitamento de medicamentos,
contribuindo para a sustentabilidade e acesso à saúde.

## Observação
Projeto acadêmico em constante evolução.
